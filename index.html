SCUTNetLogin 是专为华南理工大学设计的校园有线网络认证客户端。 支持 802.1X/EAPOL 和 DrCOM 双阶段认证，一键连接，稳定在线。

强大内核，稳定连接

从 802.1X 到 DrCOM，全协议覆盖，专为 SCUT 校园网打造

自动 IP 配置

认证前自动设置静态 IP/DNS，断连后恢复 DHCP。通过 netsh 实现，全程静默无弹窗，无需手动配置。

轻量高效

基于 Qt 6.11 + C++17 构建，多线程架构。EAP 和 UDP 分别在独立线程运行，界面响应零延迟。

开机自启

支持 Windows 任务计划自动启动，静默模式（--silent）下自动连接，托盘图标常驻，无感使用。

三步完成认证

从配置到连接，简单到只需要几次点击

1

填写配置

输入学号和密码，选择网络适配器，可选自动设置 IP 和开机自启

2

一键连接

点击"连接"按钮，自动完成 802.1X 认证和 DrCOM 心跳注册

3

保持在线

最小化到托盘，30 秒心跳自动维持，断线自动重连，无需任何操作
